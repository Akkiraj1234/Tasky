# Quick Task (GUI)

A plugin to create and edit tasks with a GUI. Stores tasks as markdown in the current open file. Compatible with Dataview, Tasks, and CardBoard plugins.

## Install / Build
1. Place this folder in `Vault/.obsidian/plugins/quick-task/`.
2. Run:
   ```bash
   cd Vault/.obsidian/plugins/quick-task
   npm install
   npm run build
